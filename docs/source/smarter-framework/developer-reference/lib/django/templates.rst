Django Templates
================

The Smarter web console is created from a Premium Bootstrap 5 based template. The theme is fully integrated
into the `Smarter base Django templates <https://github.com/smarter-sh/smarter/tree/main/smarter/smarter/templates>`__ and
can be easily customized to fit your needs.

.. important::

  Depending on the nature of your project, you might need to purchase a license for the template. Please refer to the
  `Vendor Notes`_ section below for more information.
